# Lab 1 (JavaFX)

Папка `lab_1` содержит 7 заданий по JavaFX.

## Состав

- `One.java` — задание 1 (размещение картинок на холсте, сохранение в PNG).
- `Two.java` — задание 2 (примитив в рамке, движение и изменение размера с клавиатуры).
- `Three.java` — задание 3 (добавление нескольких примитивов, стили, сохранение изображения).
- `Four.java` — задание 4 (`LineChart` для нескольких функций с настройками диапазона/шага/толщины).
- `Five.java` — задание 5 (`TableView`: добавление строк, редактирование, сортировка, скрытие столбцов).
- `Six.java` — задание 6 (3D-молекула из `.xyz`, цвет/масштаб/поворот, сохранение картинки).
- `Seven.java` — задание 7 (`Media Player`: открытие файла, play/pause/stop, прогресс, громкость).
- `sample.xyz` — пример входного файла для задания 6.
- `images/` — картинки для задания 1.

## Как запускать

Запускать каждый класс отдельно (в IDE: `Run` на нужном файле):

1. `lab_1.One`
2. `lab_1.Two`
3. `lab_1.Three`
4. `lab_1.Four`
5. `lab_1.Five`
6. `lab_1.Six`
7. `lab_1.Seven`

## Кратко по логике заданий

### 1) One.java

- Слева палитра из 3 картинок.
- ПКМ по холсту: добавить выбранную картинку.
- Кнопка `Сохранить`: снимок холста в PNG.

### 2) Two.java

- Создается случайный примитив (линия/окружность/эллипс/прямоугольник).
- Пунктирная рамка задает область примитива.
- Стрелки: перемещение.
- `+/-`: высота, `<`/`>`: ширина.

### 3) Three.java

- Выбор типа примитива слева.
- ПКМ по холсту: добавить новый примитив.
- Настройки справа: цвет линии, заливки, тип линии, толщина.
- Выбранный примитив можно двигать/масштабировать с клавиатуры.
- Меню: сохранение PNG и окно справки.

### 4) Four.java

- `LineChart` для фиксированного набора функций.
- Для каждой функции: `show` (показывать/скрывать), `width` (толщина линии).
- Общие параметры: `min`, `max`, `step`.
- Кнопка `Построить` полностью перестраивает график.

### 5) Five.java

- Таблица языков программирования (`Language`, `Author`, `Year`).
- Есть стартовые данные.
- Двойной клик по ячейке: редактирование.
- Блок `New`: добавление строки.
- Чекбоксы `Show ...`: показать/скрыть столбцы.
- Сортировка по нажатию на заголовок столбца (встроенная логика TableView).

### 6) Six.java

- Кнопка `Open XYZ`: чтение файла формата `.xyz`:
  - 1 строка: число атомов,
  - 2 строка: описание,
  - далее строки: `Element X Y Z`.
- Атомы рисуются как `Sphere` с разными радиусами/цветами по типу атома.
- Связи рисуются `Cylinder` между всеми парами атомов (простая и надежная модель связности).
- Управление:
  - `Scale` — масштаб молекулы,
  - `Rotate X/Y/Z` — поворот,
  - `Element + ColorPicker + Apply Color` — изменение цвета типа атома.
- `Save Image`: сохранение снимка 3D-сцены в `png/jpg/gif`.

### 7) Seven.java

- `Open` / `File -> Open...`: открыть медиафайл (`mp4`, `mp3`, `wav`, `m4a`, ...).
- Управление: `Play`, `Pause`, `Stop`.
- Ползунок прогресса показывает текущий прогресс и поддерживает перемотку.
- Ползунок громкости меняет `MediaPlayer.setVolume(...)`.
- Показываются:
  - имя файла,
  - текущее время проигрывания,
  - общая длительность.

## Что показать преподавателю (быстрый сценарий)

1. Открыть `Four.java`, показать выбор нескольких функций и кнопку перестроения.
2. Открыть `Five.java`, добавить строку, отредактировать ячейку, отсортировать столбец.
3. Открыть `Six.java`, загрузить `sample.xyz`, покрутить молекулу, сменить цвет элемента, сохранить изображение.
4. Открыть `Seven.java`, запустить видео/аудио, пауза/стоп, перемотка, громкость.

## Примечания

- Для задания 6 пример файла: `lab_1/sample.xyz`.
- Для задания 7 некоторые `mp4` могут не открыться из-за кодеков ОС; для демонстрации держать под рукой `mp3/wav`.
