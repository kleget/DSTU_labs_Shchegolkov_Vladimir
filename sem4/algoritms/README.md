# Алгоритмы поиска (семестр 4)

В этой папке находятся решения заданий 1-10.

## Структура

- `algorithms.py` - реализации алгоритмов и вспомогательные функции:
  - линейный поиск;
  - Бойер-Мур-Хорспулл;
  - Бойер-Мур-Хорспулл с заданным алфавитом;
  - формула (1) и rolling hash (формула (2)) для Карпа-Рабина;
  - алгоритм Карпа-Рабина.
- `experiments.py` - демонстрации и тесты для заданий 1-10.
- `main.py` - запуск всех заданий по порядку.

## Запуск

```bash
cd sem4/algoritms
python main.py
```
