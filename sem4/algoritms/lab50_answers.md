# Лабораторная 50 — графы

## 1. Выжимка теории

- **Граф** — это множество вершин и множество рёбер между ними: `G = (V, E)`.
- **Неориентированный граф**: ребро `A-B` одинаково в обе стороны.
- **Ориентированный граф**: дуга `A->B` не означает наличие `B->A`.
- **Взвешенный граф**: у ребра есть вес.
- **Смежные вершины** — вершины, соединённые ребром.
- **Смежные рёбра** — рёбра с общей вершиной.
- **Инцидентность**: вершина инцидентна ребру, если лежит на нём.
- **Степень вершины** в неориентированном графе — число инцидентных рёбер.
- **Полустепень захода** — сколько дуг входит в вершину.
- **Полустепень исхода** — сколько дуг выходит из вершины.
- **Петля** — ребро из вершины в саму себя.
- **Параллельные рёбра** — несколько рёбер между одной парой вершин.
- **Цикл** — замкнутый путь.
- **Связность** в неориентированном графе — наличие цепи между вершинами.
- **Сильно связные вершины** в орграфе — есть путь туда и обратно.
- **Односторонне связные** — путь есть хотя бы в одну сторону.
- **Слабо связные** — если убрать направления, вершины окажутся связными.
- **Матрица смежности** — квадратная матрица из 0/1 (или весов), показывающая наличие рёбер.
- **Списки смежности** — для каждой вершины хранится список соседей.
- **DFS** — обход в глубину через стек/рекурсию.
- **BFS** — обход в ширину через очередь.
- **Топологическая сортировка** применяется только к DAG — ориентированным ациклическим графам.
- **Раскраска графа** — назначение цветов вершинам так, чтобы соседние вершины имели разные цвета.
- **Хроматическое число** — минимальное число цветов, нужное для раскраски.

## 2. Ответы на 20 вопросов

1. **Что такое неориентированный граф?**  
   Граф, где ребро между `u` и `v` работает в обе стороны.

2. **Что такое ориентированный граф?**  
   Граф, где рёбра имеют направление: `u -> v`.

3. **Что такое взвешенный граф?**  
   Граф, у которого у рёбер есть числовые значения — веса.

4. **Какие вершины называются смежными?**  
   Те, которые соединены одним ребром.

5. **Какие рёбра называются смежными?**  
   Те, у которых есть общая вершина.

6. **Какая вершина является инцидентной ребру?**  
   Та, которая принадлежит этому ребру.

7. **Что такое степень вершины?**  
   Число рёбер, инцидентных вершине.

8. **Какие вершины называются связными?**  
   Те, между которыми существует цепь.

9. **Что такое сильно связные узлы?**  
   Узлы орграфа, между которыми есть путь в обе стороны.

10. **Что такое односторонне связные узлы?**  
    Узлы, между которыми путь есть хотя бы в одну сторону.

11. **Что такое слабо связные узлы?**  
    Узлы, которые были бы связны, если убрать направления рёбер.

12. **Что такое матрица смежности?**  
    Квадратная матрица, где элемент показывает, есть ли ребро между вершинами.

13. **Что такое списки смежных вершин?**  
    Представление, где для каждой вершины хранится список её соседей.

14. **Опишите алгоритм поиска в глубину в графе.**  
    Берём стартовую вершину, помечаем её посещённой, кладём в стек. Пока стек не пуст: достаём вершину, добавляем в результат, все непосещённые соседние вершины помечаем и кладём в стек.

15. **Опишите алгоритм поиска в ширину в графе.**  
    В методичке тут, скорее всего, опечатка. Обычно здесь должен быть поиск в ширину: стартовую вершину кладём в очередь, помечаем посещённой. Пока очередь не пуста: достаём вершину, добавляем в результат, всех непосещённых соседей помечаем и кладём в очередь.

16. **Что такое раскраска графа?**  
    Назначение цветов вершинам так, чтобы соседние вершины были разного цвета.

17. **Что из себя представляет топологическая сортировка графа?**  
    Это порядок вершин, при котором каждая дуга идёт слева направо: предшественник стоит раньше потомка.

18. **Для каких графов применим алгоритм топологической сортировки?**  
    Только для ориентированных ациклических графов (DAG).

19. **Опишите какой-нибудь алгоритм топологической сортировки графа.**  
    Алгоритм Кана: считаем входные степени всех вершин, кладём в очередь все вершины с входной степенью 0, потом по одной удаляем их из графа, уменьшая входные степени соседей. Если у соседа входная степень стала 0 — кладём его в очередь.

20. **Опишите какой-нибудь алгоритм раскраски графа?**  
    Жадный алгоритм: идём по вершинам по порядку, для каждой смотрим, какие цвета уже заняты её соседями, и даём ей минимальный свободный цвет.

## 4. Формат файлов для чтения/записи

### Невзвешенный граф
- строка из одного слова — отдельная вершина
- строка из двух слов — ребро/дуга

Пример:
```txt
A
B
A B
B C
```

### Взвешенный граф
- строка из одного слова — отдельная вершина
- строка из трёх значений — ребро/дуга и вес

Пример:
```txt
A
B
A B 5
B C 2
```
